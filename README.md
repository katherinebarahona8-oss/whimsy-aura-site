# Whimsy Aura / Horchata Website

Static GitHub-ready website files for the Horchata by Whimsy Aura concept.

## Files
- `index.html` — homepage
- `styles.css` — full styling
- `assets/horchata-products.png` — product rendering asset
- `assets/site-reference.png` — latest visual reference/mockup

## Deploy on Netlify
1. Create a new GitHub repo, for example `whimsy-aura-site`.
2. Upload these files to the repo.
3. In Netlify, select **Add new site → Import an existing project**.
4. Connect the GitHub repo.
5. Use these settings:
   - Build command: leave blank
   - Publish directory: `/`
6. Deploy.
